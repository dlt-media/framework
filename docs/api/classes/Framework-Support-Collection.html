Collection.php : 15

The Collection class represents a generic collection of items.

This class provides a set of methods to manipulate and interact with the collection's items, such as retrieving, setting, and removing items, as well as applying transformations and filters. It supports iteration, mapping, filtering, and retrieval of the first and last items in the collection.
